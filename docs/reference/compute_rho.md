# Compute Pairwise Correlation

Compute pairwise correlation from joint response probabilities. Based on
Dai, Ding, & Wahba (2013). Multivariate Bernoulli distribution.

## Usage

``` r
compute_rho(phi)
```

## Arguments

- phi:

  Numeric vector of length 4 with joint response probabilities.

## Value

Scalar. Correlation between outcome variables.
